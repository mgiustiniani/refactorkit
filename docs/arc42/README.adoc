== Status Conventions

* `draft` — proposed and not yet confirmed.
* `confirmed` — explicitly confirmed by user or ADR.
* `planned` — documented architectural intent not fully implemented.
* `implementation-informed` — backed by repository files, tests, or build evidence.
* `needs-review` — requires human review.
* `stale` — known to lag behind implementation.
