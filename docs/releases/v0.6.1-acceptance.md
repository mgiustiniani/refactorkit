## Purpose

`v0.6.1` is the publication patch for the completed `v0.6.0` feature scope. The
immutable `v0.6.0` tag was not published because its Windows release job failed
inside one combined clean/build/package Gradle invocation; Linux and both macOS
jobs succeeded and the publish job correctly skipped. No source tag is moved.

The patch separates clean build/test from package/native smoke in the release
workflow. Product semantics, API `0.2`, qualified toolchain and safety gates are
otherwise unchanged.

## Publication outcome

Immutable tag `v0.6.1` points to release commit
`d76adc4`. Branch CI `29290337344` passed all native targets. Tag workflow
`29291010305` again passed Linux and both macOS builders, but Windows failed in
the `clean build` invocation before packaging; publish correctly skipped.

The tag remains immutable. The clean Windows run exposed host-dependent
TypeScript glob validation that cached branch jobs had not re-executed. `v0.6.2`
fixes pattern validation while retaining clean release builds. See
[`v0.6.2-acceptance.md`](v0.6.2-acceptance.md).
