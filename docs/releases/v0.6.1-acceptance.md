## Purpose

`v0.6.1` is the publication patch for the completed `v0.6.0` feature scope. The
immutable `v0.6.0` tag was not published because its Windows release job failed
inside one combined clean/build/package Gradle invocation; Linux and both macOS
jobs succeeded and the publish job correctly skipped. No source tag is moved.

The patch separates clean build/test from package/native smoke in the release
workflow. Product semantics, API `0.2`, qualified toolchain and safety gates are
otherwise unchanged.
