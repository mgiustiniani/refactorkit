# v0.7.0 Kotlin organize-imports acceptance

Status: local library row complete; transports, packaged smoke and native matrix
remain pending.

## Qualified local row

`KotlinOrganizeImportsPlanner` accepts one authoritative non-generated saved
`.kt` file with an uncommented contiguous block of explicit non-aliased type
imports. Complete K2 usage evidence proves each import target. A directive is
removed only when its sole resolved target usage is its own import token.
Retained directives are sorted by complete trimmed directive text.

Preview emits one text edit, preserves LF/CRLF and all non-import bytes, then
recompiles the staged overlay. Apply uses `PatchEngine`; rollback restores the
original bytes.
