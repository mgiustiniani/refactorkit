# v0.7.0 TypeScript/JavaScript Workspace Index Acceptance

Status: accepted experimental provider row.

## Qualified boundary

An explicitly configured persistent TypeScript/JavaScript semantic daemon session
may contribute saved-snapshot declaration symbols to the central `WorkspaceIndex`.
The provider remains `typescript-lsp-explicit-v1`; evidence is
`language-server`, completeness is `declarations`, and mutation authority is
unchanged.

The contribution binds:

- exact project snapshot and index generation;
- TypeScript or JavaScript language partition;
- normalized source paths and exact UTF-16 declaration ranges;
- declarative TypeScript project-model projection;
- explicit Node, language-server and TypeScript toolchain evidence;
- accepted language-server name/version/capability/executable/argument hashes;
- deterministic path-free aggregate provenance hash;
- explicit truncation.

## Lifecycle

- `typescript.semantic.start` starts the provider, establishes the semantic lease,
  builds the bounded declaration projection and reports nested `index` status.
- `typescript.semantic.restart` requires the original snapshot, rotates the lease,
  revalidates provider provenance and replaces the provider partition.
- `typescript.semantic.stop`, project replacement, successful apply and session
  close remove or invalidate the partition.
- Validation or file-limit symbol-projection refusal does not terminate an
  otherwise valid diagnostics/refactoring semantic session. A request timeout or
  protocol failure stops the process and refuses startup/restart rather than
  retaining a dead semantic lease.

## Hard limits and refusal

| Boundary | Limit/behavior |
|---|---|
| Source files | 256 TypeScript or JavaScript files |
| Provider symbols | 50,000 |
| Aggregate projection deadline | 30 seconds |
| Toolchain/build evidence drift | Refused before provider request |
| Provider timeout/protocol failure | Process stopped, startup/restart refused, no dead lease or partial partition published |
| Unsafe/missing path, language mismatch or invalid UTF-16 range | Refused by `WorkspaceIndex`; no partial partition published |
| Symbol-cap overflow | Deterministic bounded prefix with `truncated=true` |

Completion, hover, signature help, position-based definition/references, editor
overlays and incremental file watching are not promoted by this row.
