## Purpose

`v0.6.2` is the publication patch for the completed `v0.6.0` feature scope. The
immutable `v0.6.0` and `v0.6.1` tags were not published because their Windows tag
jobs failed before packaging; no partial release was created and neither tag was
moved. Branch CI passed the same commits on all four native targets.

The release workflow now uses the branch-proven fresh-checkout command
`build --no-daemon`, followed by a separate package/native-smoke command. Product
semantics, API `0.2`, qualified toolchain and safety gates are unchanged.
