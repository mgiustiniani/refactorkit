## Qualified shape

The first member row accepts one public non-overloaded Kotlin function with one
compiler/class-file-proven JVM owner/name/descriptor and direct Kotlin plus Java
callers. K2 supplies exact Kotlin declaration/call tokens. The exact K2 overlay
classes let JDT resolve Java method binding tokens for the same owner and method.

The row refuses overload/bridge ambiguity, override/operator/infix shapes,
callable references, generated sources, dirty staged evidence, dynamic quoted
names and public external-consumer risk unless explicitly accepted.
